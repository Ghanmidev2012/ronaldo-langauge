# ⚽ Ronaldo Programming Language (CR7)

**The GOAT of Esoteric Languages.** A programming language built from scratch where every line of code is a goal and every output is a celebration. Forget syntax errors, here we only have "Yellow Cards".

---

## 🏟️ About the Language
Ronaldo Lang is an esoteric programming language (Esolang) inspired by the legendary **Cristiano Ronaldo**. It uses a memory-based system (The Pitch) and executes commands through the iconic **"SUUUU"** syntax.

## 🕹️ The Syntax (The Playbook)

| Command | Action | Description |
| :--- | :--- | :--- |
| `SU!` | **Goal** | Increment the value at the current position (+1) |
| `suu..` | **Foul** | Decrement the value at the current position (-1) |
| `sUu` | **Dribble Right** | Move the pointer to the next cell on the pitch |
| `UuS` | **Dribble Left** | Move the pointer to the previous cell on the pitch |
| `SUUUU` | **Celebration** | Print the ASCII character of the current value |
